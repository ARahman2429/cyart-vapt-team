# cyart-vapt-team

This repository contains week-wise Vulnerability Assessment and Penetration Testing (VAPT) reports created as part of structured cybersecurity training.

## Repository Overview

### Week 1 – VAPT Fundamentals (Practical)
- Vulnerability assessment using Metasploitable2
- Network scanning and service enumeration
- Risk analysis and remediation planning

### Week 2 – VAPT (Practical + Theory)
- Practical VAPT assessment with scanning and exploitation attempts
- Theoretical foundations of VAPT, CVSS, and exploitation concepts

### Week 3 – VAPT Capstone Project
- Full PTES-aligned VAPT engagement
- Reconnaissance, exploitation, evidence handling
- Professional reporting and management briefing

### Week 4 – Advanced VAPT (Theory)
- Advanced exploitation concepts
- Attack types, mitigations, and VAPT lifecycle

## Notes
All assessments were conducted in controlled laboratory environments for learning purposes and follow ethical hacking guidelines.
